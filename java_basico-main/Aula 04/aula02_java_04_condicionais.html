Estruturas
Condicionais

Condicionais permitem ao programa tomar decisões — executar blocos diferentes dependendo se uma condição é verdadeira ou falsa.

Fluxo de Decisão

Como o if / else funciona

O programa avalia uma condição booleana. Se for true, executa o bloco if. Senão, cai no else.

Início
condição
é true?
SIM ✓
bloco if{}
NÃO ✗
bloco else{}
Continua o programa
Sintaxe

if, else if e else

JAVA 
int nota = 75;

// if simples
if (nota >= 60) {
    System.out.println("Aprovado!");
}

// if / else
if (nota >= 60) {
    System.out.println("Aprovado!");
} else {
    System.out.println("Reprovado.");
}

// if / else if / else
if (nota >= 90) {
    System.out.println("Excelente!");
} else if (nota >= 70) {
    System.out.println("Bom!");
} else if (nota >= 60) {
    System.out.println("Regular.");
} else {
    System.out.println("Reprovado.");
}

Switch

Switch — múltiplas opções fixas

O switch é ideal quando você precisa comparar uma variável com vários valores fixos. É mais legível que uma sequência longa de else if.

JAVA — Switch.java 
int diaSemana = 3;

switch (diaSemana) {
    case 1:
        System.out.println("Segunda-feira");
        break;
    case 2:
        System.out.println("Terça-feira");
        break;
    case 3:
        System.out.println("Quarta-feira");
        break;
    case 4:
        System.out.println("Quinta-feira");
        break;
    case 5:
        System.out.println("Sexta-feira");
        break;
    default:
        System.out.println("Fim de semana!");
}
💡
Não esqueça o break! Sem ele, o Java executa todos os casos seguintes até o fim — comportamento chamado de fall-through. Na maioria dos casos, isso é um bug.
