Nível Médio — 22 min

Crie AlertaClimatico.java. Leia múltiplas variáveis climáticas e gere um boletim de alertas combinando condições com && e ||.

Variáveis a ler com Scanner:
  • Temperatura °C — double
  • Umidade % — int (0-100)
  • Velocidade do vento km/h — double
  • Está chovendo? — boolean

Alertas a verificar (cada um com sua condição composta):
  • Alerta de tempestade: chovendo && vento > 60 && umidade > 85
  • Alerta de seca: !chovendo && temp > 35 && umidade < 30
  • Alerta de neblina: umidade > 90 && temp < 15 && vento < 10
  • Alerta de calor extremo: temp >= 40 || (temp > 35 && umidade > 70)
  • Índice de Calor Aparente: sensacao = temp + (umidade / 100.0 * 8.0) - (vento / 100.0 * 3.0)

Se nenhum alerta for ativado: "✅ Condições normais"
