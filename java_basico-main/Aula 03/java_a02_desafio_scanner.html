Nível Básico — 15 min

Crie ReservaSimples.java. Use Scanner para coletar os dados de um único passageiro e emitir um bilhete de embarque.

Leituras obrigatórias (na ordem certa!):
  1. Nome completo do passageiro — nextLine()
  2. Número do CPF — nextLine() (leia como String, não número!)
  3. Número de bagagens — nextInt()
  4. Assento preferencial? — nextBoolean()

Calcule:
  • Taxa de bagagem: 1ª gratuita, demais R$89,90 cada
  • Taxa de assento preferencial: R$45,00 se true
  • Número do bilhete: int bilhete = cpf.length() * bagagens + 1000 (use a String)

Saída esperada:
╔══════════════════════════════════╗ ║ BILHETE DE EMBARQUE ║ ╚══════════════════════════════════╝ Passageiro : Ana Paula Souza CPF : 123.456.789-00 Bilhete nº : 1033 Bagagens : 2 (1 gratuita + 1 extra) Preferencial: SIM ────────────────────────────────── Taxa bagagem : R$ 89,90 Taxa assento : R$ 45,00 TOTAL TAXAS : R$ 134,90
Nível Médio — 22 min

Crie ReservaVoo.java. Leia dados de um voo completo: informações do voo + 3 passageiros, cada um com seu próprio conjunto de dados.

Dados do voo (leia primeiro):
  • Código do voo — next(), ex: "LA3045"
  • Origem e destino — next() para cada
  • Preço base da passagem — nextDouble()
  • Duração em horas — int

Para cada um dos 3 passageiros, leia:
  • Nome — nextLine() (atenção ao buffer!)
  • Classe: 1=econômica, 2=executiva — nextInt()
  • Multiplicador: econômica=1.0x, executiva=2.3x

Calcule e exiba: valor individual de cada passageiro, total do voo, média por passageiro e receita total da companhia

Teste: voo LA3045 SP→RJ, preço base=320.00, duração=1h
→ passageiro 1 econômica=R$320 | passageiro 2 executiva=R$736 | total=R$1.376
Nível Avançado — 23 min

Crie AgenciaCompleta.java. Sistema completo com menu interativo e validação de entradas.

1. Menu inicial — leia a opção com nextInt():
╔══════════════════════════════╗ ║ AGÊNCIA VAI & VEM ║ ╠══════════════════════════════╣ ║ 1. Nova reserva ║ ║ 2. Calcular seguro viagem ║ ║ 3. Cotação de milhas ║ ╚══════════════════════════════╝ Opção: _
2. Opção 1 — Nova reserva: leia nome, CPF, destino, data (como String), nº passageiros (1-9), classe
  → Valide: se nº passageiros < 1 ou > 9, exiba erro e encerre
  → Exiba comprovante com código de reserva gerado: (int)(Math.random() * 90000) + 10000

3. Opção 2 — Seguro viagem: leia destino (nacional/internacional como boolean), duração (dias) e valor da viagem
  → Nacional: 0.8% do valor da viagem + R$2,50/dia
  → Internacional: 2.1% do valor + R$5,00/dia

4. Opção 3 — Milhas: leia km do voo e categoria do cliente (char: 'P' prata, 'O' ouro, 'D' diamante)
  → Prata=1x, Ouro=1.5x, Diamante=2x os km voados
  → Se opção inválida: "Opção não existe!"

Critérios de avaliação
CPF lido como String (não int/long)
sc.nextLine() de limpeza após métodos numéricos
Menu com if/else if cobrindo opção inválida
Validação de nº de passageiros com System.exit(1)
Multiplicador de milhas com char e switch/if
Bônus: formatar CPF com printf como XXX.XXX.XXX-XX
Bônus: exibir duração do voo em horas e minutos
Checklist de entrega
Armadilhas e dicas críticas
⚠️ Armadilha principal: ao ler o int de bagagens e depois usar nextBoolean() ou nextLine(), sempre adicione sc.nextLine(); extra para consumir o \n residual.

CPF como String: CPFs com zeros à esquerda (ex: 001.234.567-89) perderiam o zero se lidos como int. Sempre use nextLine() para documentos.

→ taxaBagagem=89.90 (1 extra) | taxaAssento=45.00 | total=134.90
