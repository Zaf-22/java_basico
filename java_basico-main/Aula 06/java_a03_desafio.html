Nível Básico — 18 min

Crie Campeonato.java. Use for para processar os resultados de N rodadas fixas de um time.

Para cada rodada, leia com Scanner:
  • Gols marcados pelo time (int)
  • Gols sofridos (int)

Calcule e acumule:
  • Vitórias (gols marcados > sofridos) → +3 pts
  • Empates (gols iguais) → +1 pt
  • Derrotas → 0 pts
  • Total de gols marcados, sofridos e saldo (marcados − sofridos)

Saída esperada (N=3, resultados: 2×1, 0×0, 1×3):
╔═══════════════════════════════╗ ║ RESULTADO DO CAMPEONATO ║ ╚═══════════════════════════════╝ Rodada 1: 2 x 1 → VITÓRIA (+3 pts) Rodada 2: 0 x 0 → EMPATE (+1 pt) Rodada 3: 1 x 3 → DERROTA (+0 pts) ─────────────────────────────── Pontos : 4 Vitórias : 1 | Empates: 1 | Derrotas: 1 Gols: 3 marcados / 4 sofridos / saldo -1
Nível Médio — 22 min

Crie CampeonatoWhile.java. Agora o número de rodadas é desconhecido — use while com sentinela para ler até o usuário digitar -1 -1.

Além de acumular pontos e gols, calcule:
  • Maior goleada a favor: rodada em que o saldo foi o mais positivo
  • Pior derrota: rodada em que o saldo foi o mais negativo
  • Sequência atual: int sequencia que conta vitórias seguidas (reseta a 0 na primeira não-vitória)
  • Maior sequência de vitórias durante todo o campeonato

Teste: resultados 3-0, 2-1, 1-1, 0-2, 4-0, 3-1, -1 -1
→ maior sequência = 2 (rodadas 5 e 6); pior derrota = rodada 4 (saldo −2)
Nível Avançado — 25 min

Crie LigaCompleta.java. Sistema completo com múltiplos times e menu do-while.

1. Menu principal (do-while):
╔═══════════════════════════════════╗ ║ LIGA DAS NAÇÕES ║ ╠═══════════════════════════════════╣ ║ 1. Cadastrar time e rodadas ║ ║ 2. Ver tabela de classificação ║ ║ 3. Ver artilharia (gols/rodada) ║ ║ 0. Encerrar ║ ╚═══════════════════════════════════╝
2. Opção 1: leia nome do time e N rodadas com for. Calcule pontos, saldo e gols.
  → Suporte para até 4 times (use 4 variáveis separadas por time)

3. Opção 2 — Tabela: use for aninhado para ordenar (bubble sort manual com 4 variáveis) e exibir classificação:
Pos | Time | Pts | V E D | GM GS Saldo ──────────────────────────────────────────── 1 | Corinthians| 9 | 3 0 0 | 7 2 +5 2 | São Paulo | 4 | 1 1 1 | 4 4 0
4. Opção 3 — Artilharia: use for para calcular a média de gols por rodada de cada time e listar do maior para o menor

5. Use continue para pular times não cadastrados na exibição da tabela
Critérios de avaliação
for com número fixo de rodadas, acumulando V/E/D e gols
while com sentinela -1 -1 e rastreamento de sequência
do-while para menu persistente com opção 0 de saída
Laço aninhado para ordenar a classificação (bubble sort)
continue para pular times não cadastrados na tabela
Bônus: detectar time invicto (nenhuma derrota)
Bônus: calcular aproveitamento % (pts_obtidos / pts_possíveis × 100)
Checklist de entrega
Fórmulas e dicas rápidas
Pontuação: gM > gS → +3 pts | gM == gS → +1 pt | gM < gS → 0 pts
Sequência de vitórias: if (vitória) seq++; else seq = 0; if (seq > maxSeq) maxSeq = seq;
Bubble sort com 4 times: compare pontos de cada par com dois for aninhados, troque com variável auxiliar
Aproveitamento: rodadas * 3 = pontos possíveis; (pontos * 100.0) / (rodadas * 3) = aproveitamento %

Valores de teste básico: N=3, rodadas: 2×1 (vitória), 0×0 (empate), 1×3 (derrota)
→ Pontos=4 | Saldo=-1 | Gols marcados=3 | Gols sofridos=4
