JAVA Aula 03 — Laços de Repetição
Aula 03 · Exposição Teórica

Laços de
Repetição

Laços permitem executar um bloco de código múltiplas vezes sem repetir o código manualmente. Java tem três estruturas principais: for, while e do-while — cada uma com seu caso de uso ideal.

Visão Geral

Por que precisamos de laços?

Imagine imprimir os números de 1 a 1000. Sem laços, você precisaria escrever 1000 linhas de System.out.println(). Com um laço for, são apenas 3 linhas. Laços automatizam a repetição e são a base de algoritmos reais.

for
Ideal quando você sabe exatamente quantas vezes o laço vai repetir antes de começar.
CONTAGEM DEFINIDA
Ex: percorrer os 12 meses do ano, processar N itens lidos pelo usuário.
while
Ideal quando você não sabe quantas iterações haverá — repete enquanto uma condição for verdadeira.
CONDIÇÃO NO INÍCIO
Ex: ler entradas até o usuário digitar 0, aguardar saldo suficiente.
do-while
Garante que o bloco execute pelo menos uma vez, depois verifica a condição.
EXECUTA AO MENOS 1×
Ex: exibir menu e aguardar opção válida, jogar e perguntar se quer jogar novamente.
Laço for

Anatomia do for

O for reúne os três elementos de controle em uma única linha: inicialização, condição e incremento.

Anatomia — for ( init ; condição ; incremento )
for (int i = 0 ; i < 5 ; i++) {
    System.out.println("i = " + i);
}
Inicialização — executada UMA vez antes do laço
Condição — testada ANTES de cada iteração
Incremento — executado APÓS cada iteração
Corpo — executado se a condição for true
JAVA — for
// Contar de 1 a 5
for (int i = 1; i <= 5; i++) {
    System.out.println("Iteração " + i);
}

// Contar de trás para frente
for (int i = 10; i >= 1; i--) {
    System.out.print(i + " ");
}

// Pular de 2 em 2 (números pares)
for (int i = 0; i <= 10; i += 2) {
    System.out.print(i + " ");  // 0 2 4 6 8 10
}
🔢
Variável de controle: por convenção, usa-se i, j, k para contadores de laço. A variável declarada dentro do for existe apenas dentro dele — não pode ser acessada após o fechamento do bloco.
Laço while

while — quando não se sabe o número de iterações

O while verifica a condição antes de executar o bloco. Se a condição já for falsa na primeira verificação, o corpo nunca é executado.

JAVA — while
// Lê notas até digitar -1 (sentinela)
import java.util.Scanner;
Scanner sc = new Scanner(System.in);

double soma  = 0;
int    qtd   = 0;
double nota  = sc.nextDouble();

while (nota != -1) {
    soma += nota;
    qtd++;
    System.out.print("Próxima nota (-1 para sair): ");
    nota = sc.nextDouble();
}

if (qtd > 0)
    System.out.printf("Média: %.2f%n", soma / qtd);
⚠️
Loop infinito: se a condição nunca se tornar falsa, o programa trava. Sempre garanta que o corpo do while modifique a variável testada na condição — caso contrário, use break para sair.
Laço do-while

do-while — executa ao menos uma vez

A condição fica no fim. O bloco sempre executa pelo menos uma vez — ideal para menus que precisam aparecer antes que o usuário escolha.

JAVA — do-while
import java.util.Scanner;
Scanner sc = new Scanner(System.in);
int opcao;

do {
    System.out.println("1. Jogar  2. Regras  0. Sair");
    System.out.print("Opção: ");
    opcao = sc.nextInt();

    if      (opcao == 1) System.out.println("Iniciando jogo...");
    else if (opcao == 2) System.out.println("Lendo regras...");

} while (opcao != 0);

System.out.println("Até logo!");
Controle de Fluxo

break e continue

break encerra o laço imediatamente. continue pula para a próxima iteração, ignorando o restante do corpo atual.

JAVA — break e continue
// break — para ao encontrar o primeiro número divisível por 7
for (int i = 1; i <= 100; i++) {
    if (i % 7 == 0) {
        System.out.println("Primeiro múltiplo de 7: " + i);
        break;
    }
}

// continue — imprime só os ímpares (pula os pares)
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;
    System.out.print(i + " ");  // 1 3 5 7 9
}
💡
Uso de break com cuidado: muitos break e continue tornam o código difícil de seguir. Prefira sempre ajustar a condição do laço. Use break para casos reais de "encontrei o que procurava" ou validação de sentinel.
For Aninhado

Laços dentro de laços

Um laço dentro de outro executa o laço interno completamente a cada iteração do laço externo. Use variáveis de controle distintas (ex: i e j).

JAVA — for aninhado
// Tabuada de 1 a 5 × 1 a 5
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        System.out.printf("%4d", i * j);
    }
    System.out.println();  // nova linha ao fim de cada linha
}

// Triângulo de asteriscos
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print("* ");
    }
    System.out.println();
}
