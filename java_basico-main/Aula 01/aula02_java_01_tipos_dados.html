Tipos de Dados
e Variáveis

Variáveis são os blocos fundamentais de qualquer programa. Em Java, toda variável tem um tipo definido que determina quais dados ela pode armazenar.

Conceito Base

O que é uma Variável?

Uma variável é um espaço na memória do computador com um nome e um tipo. Em Java, antes de usar uma variável, você deve declará-la — informando seu tipo e nome.

Java é uma linguagem fortemente tipada, ou seja, você não pode misturar tipos sem uma conversão explícita. Isso garante segurança e previsibilidade no código.

JAVA 
// Sintaxe: tipo nomeDaVariavel = valor;
int idade = 25;
double altura = 1.75;
String nome = "Maria";
boolean ativo = true;
Tipos Primitivos

Os 8 Tipos Primitivos do Java

Java possui 8 tipos primitivos — são os tipos mais básicos, sem métodos ou propriedades extras.

int
Número inteiro (32 bits)
int x = 42;
double
Número decimal (64 bits)
double pi = 3.14;
float
Decimal menor (32 bits)
float f = 3.14f;
long
Inteiro grande (64 bits)
long n = 999L;
char
Um único caractere
char c = 'A';
boolean
Verdadeiro ou falso
boolean ok = true;
byte
Inteiro mínimo (8 bits)
byte b = 100;
short
Inteiro curto (16 bits)
short s = 32000;
Tipo Especial

String — O Tipo de Texto

String não é um tipo primitivo — é uma classe. Por isso começa com letra maiúscula. Ela representa uma sequência de caracteres e é uma das mais usadas em Java.

JAVA 
String curso = "Java para Iniciantes";
String vazio = "";           // string vazia
String nulo  = null;        // referência nula

// Concatenação com +
String saudacao = "Olá, " + nome + "!";
💡
Dica: Use aspas duplas " " para String e aspas simples ' ' para char. Confundir os dois é um erro muito comum de iniciantes.

Código Completo

Exemplo Prático

JAVA — Variaveis.java 
public class Variaveis {
    public static void main(String[] args) {

        // Tipos primitivos
        int     idade      = 20;
        double  nota       = 9.5;
        char    inicial    = 'J';
        boolean aprovado   = true;

        // Tipo String
        String  nome       = "João";

        // Exibindo no console
        System.out.println("Nome: " + nome);
        System.out.println("Idade: " + idade);
        System.out.println("Nota: " + nota);
        System.out.println("Aprovado: " + aprovado);
    }
}
