Nível Básico — 15 min

Crie Satelite.java. Declare variáveis fixas (sem Scanner) representando os dados de telemetria de um satélite e imprima o painel de controle.

Variáveis obrigatórias (use os tipos corretos):
  • String nomeSatelite — ex: "StarLink-42"
  • int altitudeKm — altitude em km, ex: 550
  • double velocidadeKms — velocidade em km/s, ex: 7.66
  • float temperaturaC — temperatura do painel solar em °C, ex: 126.5f
  • long orbitas — total de órbitas completadas, ex: 28_745L
  • boolean painelSolarAtivo — ex: true
  • char statusMissao — 'N' normal, 'A' alerta, 'C' crítico

Nível Médio — 20 min

Crie TelemetriaCalc.java. Mantenha as variáveis fixas do básico e adicione cálculos derivados com casting entre tipos.

Cálculos obrigatórios:
  1. Período orbital (minutos): periodo = (2 * 3.14159 * (6371 + altitudeKm)) / (velocidadeKms * 60)
  2. Velocidade em km/h: velKmh = velocidadeKms * 3600
  3. Temperatura em Fahrenheit: tempF = temperaturaC * 9.0f / 5.0f + 32
  4. Órbitas inteiras por dia: use casting int orbitasDia = (int)(1440 / periodo)
  5. Tempo total em órbita (anos): double anos = orbitas * periodo / 525_600.0

Teste com altitude=550, velocidade=7.66:
→ período ≈ 95.5 min → ~15 órbitas/dia → ~5.2 anos em órbita
