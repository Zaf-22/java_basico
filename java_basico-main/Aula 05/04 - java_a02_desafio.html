Nível Básico — 25 min

Crie CreditoBasico.java. O programa deve ler a idade e a renda mensal do cliente e verificar se ele tem direito a um cartão de crédito básico.

Regras:
✅ Aprovado se: idade >= 18 E renda >= 1.500
❌ Negado se qualquer critério não for atendido

Saída esperada:
Idade: 25 Renda: R$ 2000.00 → Cartão BÁSICO aprovado!
Dica: use sc.nextInt() para idade e sc.nextDouble() para renda. Use && para juntar as duas condições.
Nível Médio — 35 min

Crie CreditoMedio.java. Leia idade, renda e score de crédito (0-1000) e classifique o cliente em um dos três cartões disponíveis.

Regras (verifique nesta ordem):
🥇 Cartão PLATINUM: renda >= 10.000 AND score >= 800
🥈 Cartão GOLD: renda >= 5.000 AND score >= 600
🥉 Cartão BASIC: idade >= 18 AND renda >= 1.500 AND score >= 300
❌ Negado: demais casos

Teste com:
Perfil A: renda=12000, score=850 → PLATINUM
Perfil B: renda=6000, score=650 → GOLD
Perfil C: renda=2000, score=400 → BASIC
Perfil D: renda=800, score=200 → Negado
Nível Avançado — 30 min

Crie CreditoAvancado.java. Sistema completo com validação, limite de crédito calculado e mensagem personalizada.

Além da classificação do nível médio, adicione:
1. Validação: se score < 0, score > 1000, ou renda < 0 → "Dados inválidos" e encerre
2. Limite calculado:
  • PLATINUM: limite = renda * 3.0
  • GOLD: limite = renda * 2.0
  • BASIC: limite = renda * 1.0
3. Juros: se score >= 700: 1.5% a.m., senão: 2.9% a.m.
4. printf formatado com todos os dados

Saída esperada:
===== ANÁLISE DE CRÉDITO ===== Renda: R$ 12000.00 | Score: 850 Cartão: PLATINUM Limite: R$ 36000.00 Juros: 1.5% a.m. ==============================
